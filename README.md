# My Fruit Shop - TUFARM style (Next.js)

This demo fetches product data from https://api.escuelajs.co/api/v1/products and styles the UI inspired by tufarm.com.

How to run:
1. unzip and cd into folder
2. npm install
3. npm run dev
4. Open http://localhost:3000 (or 3001)

Auth:
- Register: POST to Escuelajs /users/ (form provided)
- Login: POST to Escuelajs /auth/login (form provided)
Notes:
- This is a demo using a public test API. Replace endpoints with your real backend if needed.
